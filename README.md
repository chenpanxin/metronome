# Metronome

A Simple Forum using Laravel Framework.

Inspired by [Campo Project](https://github.com/chloerei/campo) - A lightweight forum application, base on Ruby on Rails.

## Installation

```zsh
git clone https://github.com/mmruby/metronome.git

cd metronome

sudo ./setup.sh

php artisan serve
```
